# 🚖 Uber Clone - Frontend

A modern and responsive frontend for the Uber Clone application, providing a seamless ride booking experience.

---

## ✨ Key Features

- 📱 Responsive and clean UI  
- 🚗 Ride booking interface  
- 🔐 User authentication integration  
- ⚡ Fast and smooth performance  
- 🎨 Modern UI design  

---

## 🧠 Engineering Highlights

- Built using React component-based architecture  
- Integrated backend APIs for real-time functionality  
- Structured reusable UI components  
- Optimized performance and rendering  

---

## ⚡ Core Functionality

- User registration & login  
- Ride booking flow  
- Interaction with backend APIs  
- Navigation using React Router  

---

## 🛠 Tech Stack

- React.js (Vite)  
- Tailwind CSS  
- Axios  
- React Router DOM  

---

## ⚙️ Project Structure

src/
├── components/ → reusable UI components
├── context/ → state management
├── pages/ → application pages


---

## ⚙️ Setup & Run

```bash

npm run dev
