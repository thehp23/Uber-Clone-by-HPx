# 🚖 Uber Clone

A full-stack Uber Clone application built using the MERN stack, designed to simulate real-world ride booking functionality. The project demonstrates scalable architecture, authentication, and seamless frontend-backend integration.

---

## ✨ Key Features

- 🔐 User & Captain authentication (JWT-based)  
- 🚗 Ride booking and management system  
- 📡 RESTful API integration between frontend & backend  
- 📱 Responsive and interactive UI  
- ⚡ Optimized performance and smooth user experience  

---

## 🧠 Engineering Highlights

- Designed a modular full-stack architecture (frontend + backend separation)  
- Implemented secure authentication using JWT and bcrypt  
- Built REST APIs for ride booking workflow  
- Structured scalable MongoDB schemas  
- Maintained clean and reusable component-based frontend  

---

## 🛠 Tech Stack

### Frontend
- React.js (Vite)  
- Tailwind CSS  
- Axios  
- React Router DOM  

### Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT  
- bcrypt
